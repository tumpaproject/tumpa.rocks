Tumpa Landing Page

OpenPGP for smart cards made easy.

No more terminals, no more typing and entering options multiple times. All you need to do is fill up a few fields, and click a few times.

Download Tumpa

Tumpa is available as a Desktop application for MacOS (Apple silicon) and Linux

Download (v0.10.0)

Features

Easy Key Management

Tumpa enables creation and management of keys super easy. You can create, update OpenPGP keys with a few clicks using a GUI. In future you will be able to change the expiration dates, and add/revoke user ids removing any need of command line.

Tailored for Yubikeys

You can upload an OpenPGP key to a Yubikey (4/5) with minimal effort. Designed by usability researchers to maximize the user experience. Most probably the first ever desktop application which enables upload via clicks.

Built in Public Interest

We believe that privacy should be for everyone and should be an easy user experience. We are building this Free Software so that everyone can have better experience and more security using smartcards while doing OpenPGP operations.
